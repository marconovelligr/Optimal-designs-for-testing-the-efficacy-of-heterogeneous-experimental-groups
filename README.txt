README

Source code for manuscript "Optimal designs for testing the efficacy of heterogeneous experimental groups"
by A. Baldi Antognini, R. Frieri, M. Novelli, and M. Zagoraiou.

For questions, comments or remarks about the code please contact M. Novelli (marco.novelli4 at unibo.it).

The code has been written using R 4.0.4 (2021-02-15) (x86_64-w64-mingw32/x64 (64-bit)), 
running under Windows 10 x64 (build 19042)
Functions for implementing unconstrained and constrained optimal designs and their smoothed versions are provided.

IMPORTANT NOTE: Before using the functions, save the content of the code folder in the working directory so that 
		   the DLL can be properly loaded

FOR NON WINDOWS USERS:  
R CMD SHLIB conv.f
